# CNN Face Recognition System

## Overview

This project implements a CNN-based Face Recognition System using Python and OpenCV. The system detects faces from images and performs face recognition using deep learning models.

## Features

* Face Detection
* Face Alignment
* Face Recognition
* CNN-based Deep Learning Model
* Image Processing with OpenCV

## Project Structure

```text

```

## Technologies Used

* Python
* OpenCV
* NumPy
* Deep Learning
* CNN (Convolutional Neural Network)

## Installation

1. Clone the repository:

```bash
git clone https://github.com/hamsa555/hamsa_project.in.git
```

2. Navigate to the project folder:

```bash
cd hamsa_project.in
```

3. Install required dependencies:

```bash
pip install opencv-python numpy tensorflow
```

## Usage

Run the face recognition script:
python dataset.py


```bash
python recognition.py
```

## Note

Large pretrained model files are not included in this repository due to GitHub file size limitations. Download the required model files separately and place them inside the `20170512-110547` directory.
